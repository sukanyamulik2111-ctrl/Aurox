Aurora Predictor

An interactive web app where users can click anywhere on the world map to get the location's coordinates and check whether an aurora (Northern/Southern Lights) may be visible.

 Features
   Interactive world map
   Click any place to get latitude & longitude
   Aurora visibility prediction
   Simple clean interface
Tech Stack
  HTML
  CSS
  JavaScript
How to run
  Download the project and open index.html in your browser.

Future Improvements
   Real-time aurora API integration
   Better UI/UX
   Mobile responsiveness
